## 실행 방법

```bash
npm install
cp .env.example .env  # 키 채우기
node build-seo.js     # brands/ 디렉토리에 112개 HTML 생성
```

1. brands 테이블에 새 데이터 입력 시 출처 URL + verified_at 반드시 기록
2. 실제 가맹 구조(정률/정액/차액가맹금)를 구분해서 저장
3. AI 생성 수치를 "그럴듯한 값"으로 입력하지 말 것 (2026-04-24 사고의 근본 원인)
4. Supabase 대시보드 직접 입력은 git 이력 없으므로 SQL 파일로 관리 권장
5. service_role key 는 절대 채팅/로그/코드에 노출하지 말 것
