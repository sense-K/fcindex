| 파일 | 설명 |

| `build-seo.js` | 빌드 스크립트. `node build-seo.js` 한 번으로 전체 빌드 |
| `templates/brand-template.html` | 실제 빌드용 템플릿 (치환 변수 28개) |

## 실행 방법

```bash
npm install
cp .env.example .env   # .env 생성 후 키 입력
node build-seo.js       # brands/ 디렉토리에 112개 HTML 생성
```

## build-seo.js 재활용 패턴

이번에 만든 패턴은 다른 사이드 프로젝트에도 그대로 재사용 가능.

**공통 작업 순서:**
1. Supabase 테이블 준비 → `seo_enabled` 플래그 컬럼
2. `templates/item-template.html` 작성 ({{치환변수}} 구조)
3. `build-seo.js` 수정 (데이터 쿼리 + 치환 로직)
4. `sitemap.xml` 자동 생성
5. Cloudflare Pages 빌드 설정 + 환경변수 등록

**재활용 아이디어:**
- 연말정산 전략 플래너 (직장인/자영업자/프리랜서별 페이지)
- 통신비 투명성 서비스 (통신사·요금제 비교 페이지)
