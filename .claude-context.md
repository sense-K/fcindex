|---|---|

## 실행 방법

```bash
npm install
cp .env.example .env   # .env 생성 후 키 입력
node build-seo.js       # brands/ 디렉토리에 112개 HTML 생성
```

```
GitHub sense-K/fcindex (seo-rework → main)
        ↓ 자동 배포 (Build: npm install && node build-seo.js)
Cloudflare Pages (빌드 시 brands/ 112개 생성)
        ↓
undercov.kr
  /                    → index.html (기존 SPA, 그대로)
  /brands/{slug}       → brands/{slug}.html (신규 SEO 페이지)

  /robots.txt          → /community/, /admin/ 차단
```

## 주의사항

- 로컬 `.env` 파일은 절대 커밋 금지 (`.gitignore`에 포함됨)
- `brands/` 디렉토리는 git에 커밋하지 않음 — Cloudflare Pages 빌드 시 생성
- `sitemap.xml`, `robots.txt` 도 git에서 제거 예정 (빌드 생성물)
- service_role key 채팅 노출 발생 — 배포 완료 후 Supabase에서 regenerate 권장
